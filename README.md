# Data Cleaning Projects

This repository contains 5 datasets cleaned using Excel Power Query.

## Datasets

1. Customer Personality Analysis
2. Medical Appointment No Shows
3. Mall Customer Segmentation Data
4. Netflix Movies and TV Shows
5. Sales Data

## Cleaning Tasks Performed

- Removed null values
- Changed data types
- Removed duplicates
- Handled missing values
- Standardized columns
- Cleaned unwanted spaces

## Tools Used

- Excel
- Power Query
- GitHub

## Conclusion

These datasets were cleaned and transformed using Excel Power Query to improve data quality and prepare them for analysis.
